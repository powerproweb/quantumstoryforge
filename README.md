# QuantumStoryForge.io

**AetherForge Worldbuilding Hub** — steampunk science fiction worldbuilding, narrative infrastructure, interactive tools, and AI-powered writer collaboration.

Features a Steampunk Stardate Calculator, Council of 8 AI Writers Think Tank, in-universe artifact pages, and ARC team resources.

## Tech Stack

- Static HTML + embedded CSS + vanilla JS
- Steampunk dark theme with brass/gold design system
- No framework, no build step

## Deployment

Upload files directly to the server. No build step required.
