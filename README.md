# QuantumStoryForge.io

## Tech Stack

- Static HTML + embedded CSS + vanilla JS
- Steampunk dark theme with brass/gold design system
- No framework, no build step

## Deployment

Upload files directly to the server. No build step required.

QuantumStoryForge.io is a premium steampunk science fiction storyworld platform that combines AI-assisted writing tools, memory systems, and worldbuilding infrastructure in a single branded web experience. It is designed for authors, worldbuilders, and speculative fiction creators developing immersive universes, reusable lore systems, and long-arc narrative projects.

## Overview

This project presents QuantumStoryForge.io as a polished creative hub where atmosphere and functionality work together. Rather than serving as a basic content site, the platform acts as a narrative operations layer for building and navigating a fictional universe.

The site is structured around premium dark steampunk presentation, brass-and-amber visual accents, immersive hero sections, and interconnected feature pages that support deeper storytelling workflows.

## Core Experience

QuantumStoryForge.io is built to help creators:

- organize worldbuilding systems such as timelines, factions, technologies, and lore frameworks
- connect narrative tools into a unified story-development environment
- support long-form fiction, modular storytelling, and reusable intellectual property
- present story assets in a premium branded interface instead of scattered notes or disconnected pages

## Featured Modules

### Aetherforge AI Tools
- **Aetherforge AI Memory**  
  A memory-driven support layer for retaining story context, creative direction, and ongoing narrative development.

- **Aetherforge Council of 8**  
  A steampunk science fiction writers’ think tank powered by eight specialized AI personas, each designed to support storytelling, worldbuilding, and creative ideation from a distinct perspective.

### Worldbuilding Extras
- **Steampunk Stardate Generator**  
  A thematic utility for generating immersive in-universe date logic and temporal flavor.

- **HMR-9 Peregrine Field Manual**  
  A styled lore and reference page for equipment, systems, and worldbuilding documentation.

### External System
- **QD Forge Tree**  
  An external forge-tree style destination that expands the broader ecosystem connected to the QuantumStoryForge brand.
